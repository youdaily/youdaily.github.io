Python 中 ABC 模块使用

发表于 2016-12-03 分类于 编程技术 ， Python 热度： ℃ Valine： 1.2k 1 分钟

1.abc模块作用

Python本身不提供抽象类和接口机制，要想实现抽象类，可以借助abc模块。ABC是Abstract Base Class的缩写。

2.模块中的类和函数

abc.ABCMeta

这是用来生成抽象基础类的元类。由它生成的类可以被直接继承。
